# Source: https://nightwatch.laravel.com/docs/environment-variables

This page provides a comprehensive reference of all environment variables available in Nightwatch. These variables can be set in your application’s `.env` file or in the environment where your application runs.

##

Core

## 

[​](https://nightwatch.laravel.com/docs/environment-variables#sampling)

Sampling

Sampling controls the rate at which entry points are collected. For more detail, see the [Filtering](https://nightwatch.laravel.com/docs/filtering#sampling) documentation.

## 

[​](https://nightwatch.laravel.com/docs/environment-variables#data-capture)

Data Capture

## 

[​](https://nightwatch.laravel.com/docs/environment-variables#redaction)

Redaction

## 

[​](https://nightwatch.laravel.com/docs/environment-variables#filtering)

Filtering

Filtering allows you to exclude specific event types from collection. For more detail, see the [Filtering](https://nightwatch.laravel.com/docs/filtering#filtering) documentation.

## 

[​](https://nightwatch.laravel.com/docs/environment-variables#ingest)

Ingest

These variables control how the Laravel application sends data to the local Nightwatch agent.

## 

[​](https://nightwatch.laravel.com/docs/environment-variables#agent)

Agent

These variables are consumed by the standalone Nightwatch agent process, typically set in the agent’s environment or `.env` file.
